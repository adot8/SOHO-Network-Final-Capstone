# SOHO Network - Capstone Project
![Alt Text](./pics/topology.png)
## Mission
The goal of the project was to have a fully functional SOHO Network environment that included:
* Firewalls
* Routers
* Switches
* Wireless Access Point (Bonus)
* Active Directory
* Exchange Server
* Apache2 Webserver (Bonus)
* Network Redundancy
